Loop Invariants

A loop invariant is a property I that always holds at the loop entrance. For a given loop 

while (c):
    ... # loop body

we can rewrite it as 

while True:
    # l:   anything property that holds here is a loop invariant
    if not(c): break
    ... # loop body

and the loop invariant is a property I that always hold at the given location l.
